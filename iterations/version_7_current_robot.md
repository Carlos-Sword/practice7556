# Version 7 - Current Robot

## Main Idea
Current robot combines:

- Pneumatic engine
- eCVT planetary transmission
- Motion 2350 control
- Raspberry Pi camera processing
- Encoders
- Solenoid valve
- Steering servo
- Distance sensors

## Current Goals
- Reliable driving
- Stable steering
- eCVT neutral hold
- Constant carrier output control
- Camera detection
- Obstacle behavior
- Parking
