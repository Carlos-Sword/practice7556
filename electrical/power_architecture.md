# Power Architecture

## Main Power System
The robot uses a battery system to power motors, electronics, and sensors.

## Subsystems
- Motion 2350 Pro
- Raspberry Pi 3B+
- Motors
- Servo
- Sensors
- Solenoid valve
- Boost converter

## Solenoid Valve Power
The valve requires 12 V, so a boost converter is used when the main battery voltage is lower.
